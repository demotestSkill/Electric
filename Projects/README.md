# Projects — Dự án

Mỗi dự án có cấu trúc chuẩn như sau:

```
Ten_Du_An/
├── 01_Requirements/       # Yêu cầu kỹ thuật, specifications
├── 02_Design/             # Thiết kế
│   ├── electrical/        # Bản vẽ điện QET (.qet)
│   └── bom/               # Bill of Materials (.xlsx)
├── 03_Software/           # Phần mềm
│   ├── firmware/          # Firmware (C, C++, Arduino...)
│   ├── plc/               # PLC code (Ladder, ST...)
│   └── hmi/               # HMI (touch screen...)
├── 04_Manufacturing/      # Tài liệu sản xuất, lắp ráp
└── 05_Test/               # Kiểm thử, báo cáo test
```
